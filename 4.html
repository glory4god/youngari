주요태그

  1. p 태그

    단락을 나눠주는 요소! 시작과 끝을 p태그로 감싸주면 된다.

    지금처럼 줄이 띄워지는게 p태그를 이용한 효과이다!

  2. br 태그

    줄을 바꿔주는 요소 다른 태그와 다르게 시작과 끝지점을 지정할 필요가 없다
    그래서 열리는 태그하나만 있으면 되고 p태그보다 유연성이 생기며 여러개를 사용하면 지금 같이 여러번 띄울 수 있다!


    3번 br태그를 이용하면 이렇게 띄워진다!

  3. img 태그

    이미지를 사용하는 사이트는 이곳을 들어가면 무료로 사용할 수 있다.

    주요 속성에는 다음이 있다
    img src="사진파일이름" width ="사진크기(픽셀단위)" alt="사진이 깨졌을 때, 대안할 이미지" title="마우스 커서댈 때 나올 이미지 이름"
    위를 실행할 땐, 꼭 꺽새태그를 양쪽에 붙이기!

  4. table 태그

    테이블을 만드는 방법은 테이블 생성 table 태그와 테이블 행 단위로 묶어주는 tr 태그와 각각의 테이블 데이터를 지정하는 td 태그로 이루어진다.
    table 태그에 border = "표두께사이즈" 를 통해 표의 두께를 바꿀 수 있다.

    WEBprogram html css javascript

    상당히 안이쁜 테이블이다. 이런 테이블을 이쁘게 만들게 하는건 CSS에서 다룬다!

  5. form 태그

    form 태그는 입력 양식을 받는 방법으로 대표적으로 ID PASSWORD 등의 입력 정보를 받는 태그이다. 아래처럼 구현할 수 있는데 이것은 이미지를 통해서 보도록 하자!


    form action=" "은 아이디 비밀번호의 데이터를 전송하는 곳이라고 생각하면 된다.
    input엔 type 속성이 있는데 여기서 text password submit 으로 각각 텍스트형식, 비밀번호의 ***표시처리, 제출양식으로 표현하는 타입이 존재한다.
    또한 위의 학력같은 경우는 선택을 하게 되는 경우가 있는데 select를 사용하여 option을 추가하여 선택하는 방법도 추가할 수 있다.
    마지막으로 자기소개는 칸이 넓게 되어있다. 이건 textarea를 이용하면 된다!
